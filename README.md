# clone-ms-signing-client

A re-upload of Microsoft.Trusted.Signing.Client `microsoft.trusted.signing.client.1.0.95/bin/x64/Azure.CodeSigning.Dlib.dll` as a Zip file so that we can use it with `zig fetch` and pass the dll to `signtool.exe`.
